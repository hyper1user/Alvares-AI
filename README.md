# Система Генерації Рапортів ДГВ

Автоматизована система для багатомісячного обліку особового складу та генерації всіх необхідних рапортів для проходження планової перевірки.

## Функціонал

Система автоматично генерує 4 типи рапортів:

1. **ДГВ 100к** (Excel) - бійці з позначкою 100 БЕЗ "не виплачувати"
2. **ДГВ 30к** (Excel) - бійці з позначкою 30 БЕЗ "не виплачувати"  
3. **Підтвердження 100к** (Word) - бійці з позначкою 100 (навіть з "не виплачувати") + автоматичний список БР/БН
4. **Підтвердження 30к** (Word) - бійці з позначкою 30 (навіть з "не виплачувати")
5. **ДГВ 0к** (Excel) - бійці з позначкою н-п або з "не виплачувати"

## Автоматична генерація номерів БР/БН

Система автоматично генерує номери Бойових Розпоряджень за формулою:
- **Формат**: `№2025/[номер_дня_року] від [дата_попереднього_дня]`
- **Приклад**: БР на 01.05.2025 → `№2025/121 від 30.04.2025`
- **Приклад**: БР на 09.06.2025 → `№2025/160 від 08.06.2025`

## Встановлення

1. Встановіть Python 3.8 або новіший
2. Встановіть залежності:
```bash
pip install -r requirements.txt
```

## Використання

1. Запустіть головний скрипт:
```bash
python generate_reports.py
```

2. Оберіть місяць з меню
3. Оберіть тип рапорту або "Всі рапорти за місяць"
4. Система автоматично створить необхідні файли

## Структура файлів

### Вхідні дані:
- `Табель_Багатомісячний.xlsx` - головний файл з табелями за кожен місяць
- `12ШР_підтвердження_періодів_участі_у_БД_за_11.2025.docx` - шаблон Word-рапорту (опціонально)

### Вихідні документи:
- `100к_[місяць].xlsx`
- `ДГВ_30к_[місяць].xlsx` 
- `Підтвердження_100к_[місяць].docx`
- `Підтвердження_30к_[місяць].docx`
- `ДГВ_0к_[місяць].xlsx`

### Модулі системи:
- `generate_reports.py` - головний скрипт з меню
- `br_calculator.py` - розрахунок номерів БР/БН
- `excel_processor.py` - читання даних з табелів
- `word_generator.py` - генерація Word-документів
- `excel_reports.py` - генерація Excel-рапортів
- `create_template.py` - створення шаблону Excel

## Логіка роботи

1. **Читання даних**: Система читає дані з місячних листів Excel
2. **Фільтрація**: Бійці розподіляються за категоріями (100/30/0) з урахуванням приміток
3. **Генерація БР/БН**: Для кожного дня участі автоматично генерується номер БР
4. **Створення рапортів**: Формуються відповідні документи з усіма необхідними даними

## Особливості

- **Багатомісячність**: Підтримка періоду травень-листопад 2025 з можливістю розширення
- **Автоматизація**: Мінімальне втручання користувача
- **Точність**: Автоматичний розрахунок номерів БР/БН виключає помилки
- **Гнучкість**: Можливість генерації окремих рапортів або всіх одразу

## Підтримка

Система розроблена для автоматизації процесу обліку ДГВ та генерації рапортів для 12 ШР 4 ШБ 92 ОШБр.
